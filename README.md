# 🎓 SauceDemo Playwright — GitHub Classroom Шаблон

Автоматизированная система проверки студенческих проектов по автоматизации тестирования на Playwright.

## 🚀 Быстрый старт для преподавателя

### 1. Создание задания в GitHub Classroom

1. Перейдите в [GitHub Classroom](https://classroom.github.com/)
2. Создайте новое задание → "Create a new assignment"
3. Выберите этот репозиторий как шаблон
4. Настройте параметры:
   - **Deadline**: укажите дату
   - **Autograding**: включено
   - **Maximum submissions**: 10

### 2. Отправка инвайтов студентам

1. Скопируйте ссылку на задание
2. Отправьте студентам в чат/почту
3. Студенты принимают инвайт → создаётся личный репозиторий

### 3. Мониторинг прогресса

- **GitHub Classroom Dashboard**: общая статистика по группе
- **Actions tab**: детали по каждому запуску
- **Summary**: красивый отчёт после каждого пуша
